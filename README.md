### 프로젝트 개요
- 항만 현장의 유해물질을 AI로 탐지하고 안전을 강화하는 보안 로봇 프로젝트
- 선우님을 주축으로 활동
- 스마트해운물류 × ICT멘토링 2025 참여 과제로 2025년 7–8월에 수행

### 주요 기능
- 유해물질(NH3/CO/VOCs) 탐지 및 알림
- 실시간 경보 및 로그 기록
- 순찰/이동 및 영상 스트리밍(확장 가능)
